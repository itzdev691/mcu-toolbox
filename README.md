# Microcontroller Core Components Toolbox

## 📌 Description
A centralized collection of drivers, utilities, and example code for various microcontrollers, designed to simplify development, testing, and integration across multiple platforms.

## 🚀 Contents
- Hardware drivers
- Utility modules
- Example projects
- Reference implementations

## 🧩 Supported Platforms
- ESP32 / ESP8266
- Arduino-compatible boards
- STM32 (planned/partial)
- Raspberry Pi
- Other microcontrollers as added

## 🛠️ Usage
Browse the repository by platform or component and integrate the desired modules into your project. Each folder contains documentation and examples where applicable.
